What's the Difference between a HashSet and an ArrayList?

Hashset
* Duplicates are
   not allowed
* Set interface
* Unordered collection
* Object-based

ArrayList
* Duplicates are
  allowed
* List interface
* Ordered collection
* Index-based

Does Java "pass by reference" or "pass by value?"

David Flanagan, in his book, "Java in a Nutshell" puts it best: "Java manipulates objects 'by reference' but it passes object references to methods 'by value.'"
